# Daily Operations

Version: 1.0

---

# Purpose

This document defines the standard daily operational workflow for AxieOS.

The objective is to ensure that gameplay, research, and development activities are performed consistently while collecting high-quality data for future analysis.

---

# 1. Operations

## Atia Shrine

- Claim daily Fortune Slips.
- Record:
  - Streak
  - Slips before claim
  - Slips claimed
  - Slips remaining

---

## Bounty Board

- Review all six tasks.
- Reroll tasks according to the current strategy.
- Complete selected tasks.
- Record:
  - Fortune Slips spent
  - BP earned
  - Marketplace purchases
  - Marketplace sales

---

## Marketplace

Record:

- Purchases
- Listings
- Sales

---

## Wallet

Record daily balances when significant changes occur.

- WETH
- RON

---

# 3. Development

When gameplay is complete, continue AxieOS development.

Possible tasks include:

- Documentation
- Dashboard development
- Blockchain data collection
- Strategy refinement
- Repository maintenance

---

# End of Day

Generate the Daily Log only after all gameplay activities are complete.
