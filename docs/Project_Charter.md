# Project Charter

## Purpose

This document defines the mission, scope, principles, and governance of AxieOS.

## This document answers

- Why does AxieOS exist?
- What problems does it solve?
- What are the project's guiding principles?
- What is outside the project's scope?

## Audience

- Project Owner
- Contributors
- Future AI assistants
- Future developers

## Update Rules

- Update only when the project's mission or scope changes.
- Significant changes should also be recorded in the Engineering Journal.

---

# Mission

Build an open-source AI-assisted decision support system for the Axie Infinity ecosystem that helps players make better economic decisions through data, analytics, simulation, and machine learning.

---

# Vision

To become the most comprehensive GameFi analytics and decision support platform for Axie Infinity.

---

# Core Principles

1. Data before AI.
2. Measure before optimize.
3. Every recommendation should be explainable.
4. Decisions should be evidence-based.
5. Build modular, reusable components.

---

# Project Scope

AxieOS currently focuses on:

- Bounty Board
- Terrariums
- Staking
- Wallet Analytics
- NFT Portfolio
- Strategy Lab
- Machine Learning

Future modules may be added as the Axie ecosystem evolves.

---

# Out of Scope

AxieOS is not intended to:

- Automate gameplay.
- Violate the game's Terms of Service.
- Provide financial advice.
- Store sensitive credentials such as private keys or seed phrases.
